## libvirt

启动libvirtd
```
sudo apt-get install libvirt-bin
```

验证libvirtd是否启动
```
sudo systemctl is-active libvirtd

active
```

安装libvirt
```
apt-get install libvirt-dev libvirt-daemon libvirt-clients
```
